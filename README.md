# Hindi → English Vegetable Translator (Beginner Python Project)
A super beginner-friendly Python project that converts **Hindi vegetable names → English names** using a clean dictionary-based mapping.  
Perfect for students learning Python, pandas, and simple language translation concepts.
## Features
-  Hindi → English translation (manual dictionary mapping)  
-  Clean & simple Python notebook  
-  Uses pandas for loading, saving, and mapping data  
-  Great for NLP & data processing beginners  
-  Easy to extend with more words or translation APIs  
## What You Learn From This Project
- How to build your own **language mapping dictionary**  
- How to use **pandas DataFrame mapping**  
- How to load & save CSV files  
- How to create small translation datasets  
This makes it a great **mini-project for resumes, GitHub profiles, and college submissions**.
